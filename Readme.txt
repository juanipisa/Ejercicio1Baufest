Para ejecutar la pruebas se debera abrir el proyecto EjercicioBaufest con Eclipse y ejecutar dependiendo de la prueba que se quiera realizar 
cualquiera de los Features creados para estas solamente presionando en la flechita al lado de Run
Run Configurations -> Cucumber Features -> (Seleccionar feature)

El proyecto esta compuesto por 4 features:
- cerrarSession.feature
- agregarLaptop.feature
- iniciarSesion.feature
- registroScenario.feauture

Se utilizo Java como lenguaje de programacion, Cucumber y Maven.
